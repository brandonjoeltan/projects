How degradation tracking works

Normalization: each test is divided by either the average of control before/control after, or by the single control selected if only one is provided.
normalized test = test / average(control before, control after)
or
normalized test = test / selected control
Bulb type tracking: assign the same bulb type label to related files, such as H bulb, D bulb, prototype A, Batam, or Gaithersburg.
Degradation chart: the app groups normalized tests by bulb type and plots the selected metric against life-tested hours.
Excel append: import a prior workbook from this app, and its degradation sheet will be merged into the current degradation chart and table.
