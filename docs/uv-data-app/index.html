UV Spectral Data Dropzone

Drop one or more spectrometer .txt files, set the distance in cm, rename each output column, assign each file as either a Control or Test, and for each test select the control measured before and/or after. If both are selected, the app averages them wavelength-by-wavelength. If only one is selected, the app uses that single control directly. The test is then normalized against that control reference.

normalized test = test / selected control
Option C: leave both blank.
the test will stay unnormalized and will not appear in normalized outputs
